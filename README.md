### Описание
Проект представляет собой парсер вакансий с сайта: https://www.sql.ru/forum/job-offers. 

### Использованные технологии
* JDBC
* PostgreSQL
* Библиотека Jsoup
* Collections Framework
* Sockets
* Checkstyle plugin
* JaCoCo
* Travis CI
